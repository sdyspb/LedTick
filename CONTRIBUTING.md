# Contributing to LedTick

## Reporting Issues

- Check existing issues first.
- Describe the problem and steps to reproduce.
- Add photos or logs if possible.

## Pull Requests

1. Fork the repo.
2. Create a descriptive branch.
3. Make your changes.
4. **Test manually on your device.**
5. Open a Pull Request explaining what you changed and why.

## Questions

Open an issue or contact me.
